# Agent Rules

## 작업 흐름 (모든 작업에 필수 적용)

작업 1건당 아래 순서를 반드시 따른다.

- 1단계: `doc/todo_작업명.md` 문서 생성 — 아래 양식 사용
- 2단계: 실제 작업 수행
- 3단계: 작업 결과를 문서에 반영
- 4단계: 커밋 후 푸시

### todo 문서 양식

```
## 작업 내용

## 작업 결과
```

---

- `~/` 유저 홈 설정을 따르지 않는다.
- 모든 문서는 한국어로 작성한다. 최대한 한국어를 사용한다.
- 한자가 언급되는 경우, 모든 한자에 대한 한국어 설명을 추가한다.
- 표(table) 형태 대신 목록(list) 형태를 사용한다.
- 커밋은 `git commit -m "메시지"` 형태로 간략하게 진행한다. cat 또는 HEREDOC을 사용하지 않는다.

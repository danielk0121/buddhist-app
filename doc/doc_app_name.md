## 앱 이름 선정

```markdown
프로젝트 성격을 고려한 후보들입니다.
읽기 중심

경독(經讀) — 경전을 읽다, 짧고 기억하기 쉬움
법문소(法文所) — 법의 글이 있는 곳
수경(隨經) — 경전을 따라가다

수필·이야기 강조

경필(經筆) — 경전 + 수필의 합성, 발음도 자연스러움
불설(佛說) — "부처님이 말씀하시길", 원문 느낌
법담(法談) — 법에 대한 이야기, SNS 소통 느낌도 살림

현대적·접근성 강조

법구(法句) — 법의 문장, 법구경에서 따온 이름
일불(一佛) — 누구나 하나의 불성, 심플한 브랜딩
연화(蓮花) — 진흙 속 연꽃, 어려운 불경을 쉽게 핀다는 의미


개인적으로는 경필(經筆) 을 추천합니다. 
경전(經)과 수필(筆)이 자연스럽게 합쳐지고, 한자 두 글자라 앱 아이콘·도메인(gyeongpil.com)으로도 깔끔합니다. 
프로젝트 콘셉트를 이름 하나로 설명할 수 있다는 것도 장점이고요.
```

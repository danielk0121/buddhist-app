## 작업 내용

북마크 목록 화면 구현 — Phase 4 준비 항목

- 기획서 4.5 항목 기반
- 북마크 저장 기능은 이미 구현됨 (`webapp/api/dummy/bookmark.js`)
- 저장된 북마크를 볼 수 있는 화면이 없으므로 추가 구현 필요
- 구현 범위
  - 북마크 목록 화면 (`webapp/pages/Bookmarks.jsx`) 신규 생성
    - localStorage에서 북마크 slug 목록 조회
    - 북마크한 경전 카드 목록 표시 (제목, 한자, 계열, 이모지)
    - 북마크 해제 버튼 (각 카드에 ✕ 또는 ★ 토글)
    - 북마크 없을 시 안내 문구 표시
    - 경전 카드 탭 시 경전 상세로 이동
  - 설정 화면 (`webapp/pages/Settings.jsx`) 수정
    - 북마크 목록 화면으로 이동하는 버튼 추가
  - 라우트 추가 (`webapp/App.jsx`)
    - `/bookmarks` 경로 추가
    - 탭바 숨김 없이 표시 (탭 화면 아님 — 서브 화면)

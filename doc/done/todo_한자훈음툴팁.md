## 작업 내용

한자 훈음(訓音) 인라인 툴팁 UI 구현 — Phase 1 미완료 항목

- 기획서 4.1, Phase 1 항목 기반
- 경전 상세 본문에서 한자(漢字) 또는 불교 용어를 탭/클릭 시 훈음·해설 툴팁 표시
- 구현 범위
  - 경전 본문 내 한자 감지 및 `<ruby>` 또는 커스텀 툴팁 컴포넌트 적용
  - 단어 사전 단락의 항목을 기반으로 툴팁 데이터 연결
  - 모바일 터치 및 데스크톱 마우스 호버 양쪽 지원
- 툴팁 표시 내용: 한자 + 훈음(독음) + 간단한 한국어 해설

## 기획서 초안에 내용 추가
- 상단 툴바 사용
- 주요 요구 사항
    - 경전 수필 목록 보기
    - 경전 수필 상세 보기
    - 개인 설정 화면
    - 개발자 정보 화면 (저작권, 오픈소스 고지 화면 링크 포함)

기획서 문서를 기준으로 더 구현해야 하는 웹앱 기능이 있는지 검토

---
기획서 > tts 오디오북 기능 추가 > 웹앱 메인 메뉴에 추가
웹앱 > 상단 툴바 > 경전, 오디오북, 설정 > 3개 탭바로 구성되도록 수정 > 기획서 내용 추가 > 작업 > 커밋
경전 > 메인 화면을 타일 형태로 배치 (ref 폴더에 참고 이미지 반영)
앱 전체 > 이모지 삭제

---
우상단 목록 > 햄버거 버튼 사용
인기 경전 기능 삭제
전체 경전 > 전체 보기 버튼 삭제 > 항상 모든 경전을 목록에 표시 (50개가 안됨)
경전 홈 화면 > 각 경전 타일 구조의 각 타일의 화면 > "이모지(줄바꿈)제목" 을 가운데 배치해서 이미지인 것처럼 사용

---
설정 > 언어 변경 > 한국어, 영어 지원
경전 홈 > 각 경전 타일 > 번호 부여
재생버튼 > 이모지 아이콘 사용 > 각각의 이모지가 겹치지 않도록 사용
오디오북 > 경전 홈 화면과 동일하도록 타일형태 사용

---
